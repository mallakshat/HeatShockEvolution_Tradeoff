# HeatShockEvolution_Tradeoff
This repository contains data and scripts for the paper titled - "Evolution uncovers a general tradeoff between recovery after heat shock and growth at elevated temperatures", published in _mBio_ - [https://doi.org/10.1101/2025.10.22.683249.](https://doi.org/10.1128/mbio.03305-25)

This work focuses on adaptation to heat shock, using _Methylobacterium extorquens_ as a model organism. There are two major aspects to this work - 
1. Evolution experiments to understand the genetic basis and degree of parallelism in adaptation to heat shock.
2. Understanding the correlation between fitness in high temperature regimes of different intensities. 

The majority of the work is empirical and the datasets generated and scripts used for analysis are stored in this repository. All scripts to analyze the data and make figures are included in the MainText_Code and Supplementary_Code folders respectively, based on whether the code is used for figures in the main or supplementary text. For example, "FigS2_Evolved_Viability.R" in the Supplementary_Code folder refers to the script used for the analyzing data and making the plot associated with Fig S2 in the Supplementary Material. The experimental data used for the analysis is available in the corresponding Supplementary_Data folder.
